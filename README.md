# Telegram Subscription Bot

A scalable Telegram bot for managing paid subscriptions to private channels.

## Features
- Multi-channel subscription management
- UPI QR payment flow
- Admin approval/reject system
- Validity tracking with stackable renewals
- Auto expiry cleanup + renewal reminders
